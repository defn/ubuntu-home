    Save the one-time exported gpg key:

    find .gnupg/sk*

    Generate the public gpg key:

    gpg2 --armor --export $email

    Generate the ssh key from the gpg key:

    ssh-add -L

    Change yubikey pin:

    gpg2 --card-edit
